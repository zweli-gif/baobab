## Responsive Behavior

### Desktop (>1024px)
- Single-column layout, non-scrolling
- All sections visible at once
- Bottom nav hidden

### Tablet (768-1024px)
- Single-column layout, scrollable
- All sections stacked
- Bottom nav visible

### Mobile (<768px)
- Single-column layout, scrollable
- All sections stacked
- Bottom nav visible (fixed)
- Padding reduced to 16px
- Font sizes reduced by 10%
